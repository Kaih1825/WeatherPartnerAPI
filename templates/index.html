Weather Partner API

Weather Partner API 文件

歡迎使用Weather Partner API 文件！這個 API 允許您與天氣資料進行交互。

Base URL

https://weatherpartner.tplinkdns.com/

端點

描述： 將天氣資料發送至伺服器。

請求主體：

  • `location`（字串，必填）：位置名稱

回應：

{
  "message": "User success",
  "uuid": "<generated UUID>"
}

描述： 透過 UUID 更新天氣資料。

路徑參數：

  • `uuid`（字串，必填）：資料的唯一識別碼

請求主體：

  • `temp`（字串，必填）：溫度
  • `wet`（字串，必填）：濕度
  • `purple`（字串，必填）：紫外線指數
  • `water`字串，必填）：是否正在降雨

回應：

{
  "message": "User success",
  "uuid": "<updated UUID>"
}

描述： 獲取所有天氣資料。

回應：

{
  "data": [
    {
      "uuid": "<UUID>",
      "location": "<location>"
    },
    // 更多資料...
  ]
}

描述： 透過 UUID 獲取天氣資料。

路徑參數：

  • `uuid`（字串，必填）：資料的唯一識別碼

回應：

{
  "data": [
    {
      "uuid": "<UUID>",
      "time": "<timestamp>",
      "location": "<location>",
      "temp": "<temperature>",
      "wet": "<wetness>",
      "purple": "<purple-level>",
      "water": "<water-level>"
    }
  ]
}

描述： 透過 UUID 刪除天氣資料。

路徑參數：

  • `uuid`（字串，必填）：資料的唯一識別碼

回應：

{
  "message": "資料與 UUID '<uuid>' 已成功刪除"
}
